#13 — Increase text size on small screens

Current: class="block rounded px-3 py-2 text-sm font-medium no-underline"

Fix: Use max-sm:text-base (or appropriate Tailwind responsive prefix) to increase font on smaller viewports without custom media queries.

text-sm max-sm:text-base

Also apply same to the trigger label span.

#11 — Center the dropdown itself

Current trigger: In a relative container — trigger button is left-aligned by default.

Fix: Add justify-center to the parent flex container to center the trigger button. Change dropdown positioning from left-0 to left-1/2 -translate-x-1/2. 

<div class="relative flex justify-center sm:hidden">
    <div class="mobile-nav-toggle flex cursor-pointer items-center gap-2 rounded-md px-2 py-1">
        <span class="text-sm font-medium">{currentPageLabel}</span>
        <svg>...</svg>
    </div>

    <!-- Dropdown centered under trigger -->
    <div class="mobile-nav-dropdown absolute top-full left-1/2 z-50 -translate-x-1/2 mt-2 ...">

#15 — Fix triangle animation persistence

Root cause: The SVG triangle's rotate-180 class is applied but may not work due to missing transform-origin or the class being overridden. The <path> uses absolute coordinates (M1 1L5 7L9 1) so rotation origin matters.

Fix: Add transform-origin: center via Tailwind (origin-center) and ensure the transition targets transform properly. Alternative: wrap the SVG in a <span> with the rotation class for more reliable behavior.
